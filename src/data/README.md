# Data

```
data/
├── raw/         # source files migrated from the gitignored "Walk On EMA" Teams folder
│   ├── baseline_begin_studie.sav     # baseline questionnaire (POAM-P, demographics, ...)
│   ├── ema_triggerlevel_merged.xlsx  # trigger-level EMA + Axivity ENMO (one row per prompt)
│   ├── diary_dagboek.sav             # morning/evening diary (not used in current pipeline)
│   └── barriers.sav                  # barriers questionnaire (not used in current pipeline)
└── processed/   # analysis-ready outputs of stage 01 (regenerated by the pipeline)
    ├── ema_long.csv          # one row per trigger, analytic sample (N=34, 1894 rows)
    ├── person_level.csv      # one row per participant (POAM-P, demographics, determinants)
    ├── exclusion_log.csv     # audit trail of excluded participants + reasons
    ├── analytic_tokens.csv   # token <-> anonymous pid map for the 34 analytic participants
    ├── poamp_subgroups.csv   # POAM-P dominant pattern + cluster labels (stage 08)
    └── link_analysis_frame.csv  # per-person network params + POAM-P (stage 09)
```

Raw clinical data are **not** committed (see project `.gitignore`). The variable codebook
is in `src/utils/study_materials/CODEBOOK.md`. Everything in `processed/` is regenerated by
`python src/utils/pipeline/full/run_all.py`.
